# Hugo CLI


### Commands

**Create new site**
###### Syntax
`hugo new site [path] [flags]`

###### Example
```
$ hugo new site <sitename>
```
[more...](https://gohugo.io/commands/hugo_new_site/)  

**Create new page**
```
$ hugo new {route/}<pagename>
```

**Serve site in localhost**
```
$ hugo server <-D>
```
